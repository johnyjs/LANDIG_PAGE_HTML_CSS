Balle Bot

Deixe a sua comunidade de jeito fácil

O que ela faz por você ?

1. Cria um sistema de verificação capcha para novos menbros garatirem que não são robôs

2. Possuem um sistema de banimentro automático para links suspeitos que forem enviados para comunidade

3. Sistema de avisos de banimento manual, para pessoas administradoras ultilizarem
